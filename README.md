# :soccer:FootballFanApp 
Simple application to check latest scores and upcoming matches in 7 Top Soccer Leagues.

Build with API data from https://www.football-data.org/

App is using free-limited version with 10calls/minute and limited leagues.

# :key:Technologies
Project is written in **Kotlin** with usage of:

* MVVM architecture
* Kotlin Navigation
* Dagger - Hilt
* ROOM 
* Coroutines
* Retrofit 2
* Coil
* Gson
